# Extenso
Escreve números por extenso com GoLang

![Go](https://github.com/joaomannes/extenso/workflows/Go/badge.svg)

## Usage

```go

package main

import (
	"fmt"

	"github.com/joaomannes/extenso"
)

func main() {
	fmt.Println(extenso.From(1234))
}
```

## Instalation
```
go get github.com/joaomannes/extenso
```
